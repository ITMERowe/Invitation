# Invitation — Wedding Invitation

A small set of HTML tools to edit and preview a wedding invitation (English + Indonesian).

Contents
- `index.html` — The wedding invitation website (view in browser).
- `dashboard.html` — Dashboard UI for editing content and settings per-section (names, date, venue, rundown, dress, RSVP) with side-by-side English/Indonesian editing.

Quick start
1. Open `index.html` in your browser to view the invitation.
2. For advanced editing per section, open `dashboard.html`:
	- Use **Open index.html** to load the site.
	- Edit sections (Names, Date, Venue, Rundown, Dress, RSVP) and click **Save This Page** to apply changes in memory or save directly when the browser supports File System Access.
